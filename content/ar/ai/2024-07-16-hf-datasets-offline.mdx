هذا الأمر بسيط جدًا، لكنه مفيد للغاية إذا كنت تشغّل مهامًا على عقدة حوسبة لا يتوفر فيها اتصال بالإنترنت.

على عقدة تسجيل الدخول أو على جهاز آخر متصل بالإنترنت، شغّل كود Python التالي:

```python
import datasets

x = datasets.load_dataset("my_dataset")

x.save_to_disk("./my_dataset_local")
```

بعد ذلك، إذا لزم الأمر، انسخ الملفات إلى الجهاز الذي تُشغِّل عليه مهمتك. ومن ذلك الجهاز غير المتصل بالإنترنت، يصبح تحميل مجموعة البيانات بسيطًا!

```python
y = datasets.load_from_disk("./hellaswag_local")
y # DatasetDict({...})
```
