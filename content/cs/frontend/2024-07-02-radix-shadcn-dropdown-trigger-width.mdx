Na svém webu používám [shadcn Dropdown Menu](https://ui.shadcn.com/docs/components/dropdown-menu) (postavené na Radix `DropdownMenuPrimitive`) k tomu, aby si uživatelé mohli změnit aktuální motiv webu.

Ve výchozím nastavení nemá obsah dropdown komponenty stejnou šířku jako její spouštěč. Vadilo mi to, a tak jsem nějakou dobu hledal řešení. Nakonec jsem našel odpověď [v dokumentaci Radixu](https://www.radix-ui.com/primitives/docs/components/dropdown-menu#constrain-the-contentsub-content-size)!

Vložte níže uvedený kód do souboru CSS (pokud nepoužíváte Tailwind, můžete obal `@layer utilities` odstranit):

```css
@layer utilities {
  .dropdown-content-width-full {
    width: var(--radix-dropdown-menu-trigger-width);
    max-height: var(--radix-dropdown-menu-content-available-height);
  }
}
```

Nyní upravte svůj `<DropdownMenuComponent>` tak, aby vypadal takto:

```jsx
<DropdownMenuContent align="end" className="dropdown-content-width-full">
```

Voilà! Obsah i trigger mají stejnou šířku. Není zač.
