# bengubler.com

Welcome to my personal website!

**Built with:**

- https://v0.dev (UI design)
- Next.js 16 + React 19
- Tailwind CSS v4
- TypeScript
- Content Collections + `content-pipeline` (Markdoc + Shiki)
- shadcn/ui + Radix UI
- Turbopack dev builds

**Features:**

- Dark mode
- View transitions
- Giscus comments
- RSS feeds
- Reading time + auto TOC
- Validated custom content tags
